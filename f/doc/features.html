Symbol: Fivespot - Nemophila-maculata

  • Reliability.
  • Portability.
  • Speed of Execution.
  • Speed of Development.
  • Well Documented.

Features.

  • Syntatic delegation.
  • Easy but flexible transparent persistence.
  • Object query language.
  • All objects with default to_json and to_xml methods implementations.
  • Factory provided by language allowing to new instantiation, singleton, pool and mockups.
  • Interface implemented just by implementing the methods, no need to declare a class implements it. The compiler should know.
  • Minimalist make files, just point to the main module and search folders and all dependencies are calculated and searched for in the current and children folders by default and on the list of search folders provided.
  • Global code insertion with flexible description language.
  • Logging ready, easy and configurable.
  • Internationalization ready and easy. Strings are internationalized by default. Use directive i18_off{} to disable it for the enclosed code.
  • Class attributes are read and write by default. Implement Get and Set if any special processing is needed. Use r to specify read only, w for write only, rw (default) for read/write attributes.
  • Every object descends from Object class.
  • As of date logic ready to use.
  • Context sensitive inteligent parser.
  • Auto generated Metadata, Mockup and Test classes.
  • uses separate from inherit. Mixins -> use. Derived classes -> inherit.
  • CRUD operations create automatically for all objects not marked with crud_off.
  • Operator overloading.
  • Deterministic controllable garbage collector with subscribable events (OnBeforeDestroy, OnAfterDestroy).
