Tokens

Token Ids are prime numbers in order to be mathematically possible to deduce the tokens for a given expression id which is formed using the same rules as Godel symbol numbers.

;

Separates expressions in a line.
Token ID: 1

.

Separates object instance from member access or member function call.
Token ID: 2

+

Addition binary operator.
Token ID: 3

-

Subtraction binary operator.
Token ID: 5
