CSK keywords.

Note: In alternatives the bold and blue one indicates default value.

C - Class

^[ \t]*C[ \t]+([a-zA-z0-9]{1,255})[ \t]*([ \t]+((private|inheritance|public)[ \t]+)*((u|i)[ \t]+([a-zA-z0-9]{1,255}))*)*$

Examples:

C Vehicle
{
    I maker_id
    S model
    Color vcolor
}
C Car i Vehicle
{
    I passenger_doors_count
}

E - Enum

E <enum_name> {<enum_item0>, <enum_item1>}

Examples:

E ContactInfoQualifier {ciHome, ciWork, ciOther}

u - Uses


Examples:

C Tax u AsOfDate
